Layer 2 · Agent Orchestrator — Role Enforcement & Task Execution Loop
Role Enforcement
1. Read persona's agent.yaml
2. Check: is EDR or RRO allowed?
3. Enforce approval gate (Auditor)
4. Inject role context into session
5. Load use case skill file
⛨ Queue Analyst → EDR ✓
⛨ Queue Analyst → RRO ✓
⛨ Auditor → EDR ✓ (approval)
⛨ Auditor → RRO ✗ denied
Execution Engine — Task Loop
Reads the use case YAML planner, iterates each task, resolves skill + tool, executes via LLM, moves to next task
For each task in use case planner:
1Load task definition from UC YAML
2Resolve task.skill_ref → pull from skill registry (L1/L2/L3)
3Resolve task.tool_ref → pull from tool registry
4Compose: job desc + skill context + tool
5Send to Agent LLM Worker → execute
6Return result → proceed to next task
